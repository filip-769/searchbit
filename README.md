## Features
- 27 search engines
- Web & image search
- Instant answers
- No JavaScript required
- Cookies used only for settings
- 58 UI themes
- HTTP proxy option
- Convert html to jpeg and docx, pptx to pdf
- Web proxy
- Strong CSP preventing IP leak
- JSON config
